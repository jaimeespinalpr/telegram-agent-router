Telegram Agent Router

A FastAPI and Telethon router that lets one Telegram account coordinate a director bot, worker bots, and human helpers through a small web control panel.

View repository Run locally
Chatty Director @chattydabot receives the owner order and dispatches structured tasks.
Worker Bots @multida_bot, @ardidabot, and @prpagydabot receive delegated tasks.
Live Backend The Telegram listener runs on a server or local machine, not on GitHub Pages.

Deployment Note

GitHub Pages hosts this static project page. The actual router requires Python, Telethon session storage, and environment variables, so run it with uvicorn app.main:app --reload or deploy it to a backend host.
